# nasz_lokal 

Aplikacja napisana w django z frontendem HTML/CSS/JS. Aplikacja służy dodawaniu pomysłów na restaurację pogrupowanych na kategorie, które także można dodawać. Wpisy można edytować i usuwać. Zastosowałem także proste animacje i skrypty JS. Dodatkowo jest też autoryzjacja użytkownikow.

Aplikacja jest skończona i wykonałem jej deployment na serwerze Linux, obecnie testujemy ją w środowisku produkcyjnym. Aplikację napisałem pod kątem zbierania pomysłów na lokal jaki chcę otworzyć z dziewczyną w przyszłośći.
